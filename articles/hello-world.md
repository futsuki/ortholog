これは **Simple Blog** の最初の記事です。

## 使い方

1. `admin/` にアクセスして管理画面にログインします（デフォルトパスワード: `password`）
2. 管理画面から `.md` ファイルをアップロードします
3. 記事は自動的に一覧表示されます

## Markdown 記法

- **太字** や *斜体* が使えます
- リストも書けます
- [リンク](https://example.com) も貼れます
- `コード` も ```` ``` ```` で囲めます

> 引用もできます!

## テーブル

| 機能 | 説明 |
|------|------|
| 太字 | `**text**` |
| 斜体 | `*text*` |
| リンク | `[text](url)` |
| コード | `` `code` `` |

Enjoy blogging!
